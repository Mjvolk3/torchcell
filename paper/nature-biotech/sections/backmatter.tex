\bmhead{Supplementary Note~1: functional equivalence of the perturbation operator}

Figure~\ref{fig:torchcell}g and the Methods state that applying the perturbation operator
to the fixed reference representation approximates re-encoding the rebuilt perturbed graph,
$\Phi(G_p)\approx\mathcal{R}_\phi(\mathcal{T}_\psi(H,p))$. This note makes the statement
precise: the reparameterisation is an \emph{exact} identity, whereas the trained
approximation is a \emph{learned} correspondence with explicit realisability conditions.
Notation follows Table~\ref{tab:notation}; two symbols are local to this note---the
\emph{perturbed graph} $G_p$ and the deterministic \emph{rebuild map}
$\kappa:(G,p)\mapsto G_p$ that materialises it (we write $\kappa$, not $\rho$, because
$\rho$ already denotes the gene--reaction association). $\Phi,\Psi$ denote arbitrary
measurable target maps, $D$ the data distribution over $(G,p)$, and $\sigma(\cdot)$ the
generated $\sigma$-algebra.

\paragraph{Two ways to predict a phenotype under a perturbation $p$.}
\emph{Perturb the data (graph network).} Materialise $G_p=\kappa(G,p)$ by deleting or
rewiring vertices and edges, then encode, $\hat y=\Phi(G_p)$; the graph is a hard
constraint and $\kappa$ runs once per strain. \emph{Perturb the representation (CGT).}
Encode once and operate,
$\hat y=\mathcal{R}_\phi\!\big(\mathcal{T}_\psi(F_\theta(G),p),\varepsilon\big)$; the graph
enters only through $F_\theta$, which runs once, and $H$ is reused across all $p$.

\paragraph{Proposition (reparameterisation).}
Let $\mathcal{H}_{\mathrm{data}}=\{\Phi\circ\kappa\}$ be the maps expressible by perturbing
the data and
$\mathcal{H}_{\mathrm{rep}}=\{\Psi:\mathcal{G}\times\mathcal{P}\to\mathcal{Y}\}$ those
expressible as a function of $(G,p)$ directly (the amortised CGT family). Then
\[
\mathcal{H}_{\mathrm{data}}\subseteq\mathcal{H}_{\mathrm{rep}},
\]
with equality iff $\kappa$ is injective on $\operatorname{supp}D$. The two learning problems
therefore target the same function class, and the amortised family is strictly larger
exactly when the rebuild destroys information.

\paragraph{Proof.}
\emph{Containment.} $\kappa$ is deterministic, so $(G,p)$ fixes $G_p$. For any target
$\Phi^\star$, $\Phi^\star(G_p)=(\Phi^\star\circ\kappa)(G,p)=:\Psi^\star(G,p)$---an equality,
giving $\mathcal{H}_{\mathrm{data}}\subseteq\mathcal{H}_{\mathrm{rep}}$. \emph{Strictness.}
The converse requires $\sigma(G_p)=\sigma(G,p)$. Rebuilding is information-destroying: $p$ is
generally not recoverable from $G_p$, so $\sigma(G_p)\subsetneq\sigma(G,p)$ when $\kappa$ is
non-injective, and some $\Psi\in\mathcal{H}_{\mathrm{rep}}\setminus\mathcal{H}_{\mathrm{data}}$
exists.\hfill$\square$

\paragraph{Concrete example.} Three genes with base graph $G$ the path $A\!-\!B\!-\!C$.
\emph{Injective case.} Deleting $B$, $p=\{(g_B,\mathrm{del})\}$, drops $B$ and its edges, so
$G_p=\{A,C\}$ (both isolated); the target $\Phi(G_p)=\#\mathrm{edges}=0$ is reproduced
without any rebuild by
$\Psi(G,p)=\#\mathrm{edges}(G)-\#\{\text{edges incident to }B\}=2-2=0$.
\emph{Non-injective case.} Overexpressing $B$ at two doses,
$p_1=\{(g_B,\mathrm{OE},1)\}$ vs $p_2=\{(g_B,\mathrm{OE},3)\}$, changes no topology, so
$\kappa(G,p_1)=\kappa(G,p_2)=G$; any $\Phi(G_p)$ returns one value for both, yet
dose-response phenotypes differ. The pair $(G,p)$ still carries the magnitude $m\in\{1,3\}$
and separates them---continuous dose, and likewise the perturbation type $\tau$, lives in
$p$, not in a bare rebuilt topology. This is
$\mathcal{H}_{\mathrm{rep}}\setminus\mathcal{H}_{\mathrm{data}}$ made concrete.

\paragraph{Corollary (origin of the $\approx$).}
The ideal amortiser is $\Psi^\star=\Phi^\star\circ\kappa$. CGT realises it not by that
literal composition but by the learned $\mathcal{R}_\phi\circ\mathcal{T}_\psi$ acting on
$H=F_\theta(G)$. If (i)~$F_\theta$ is injective on $\operatorname{supp}D$, so $(H,p)$ carries
the same information as $(G,p)$---free for a fixed gene set, since $H$ keeps one identifiable
row per gene---and (ii)~cross-attention over the perturbation set followed by an MLP readout
is a universal approximator of set-conditioned permutation-equivariant maps, then for every
$\epsilon>0$ there exist $\psi,\phi$ with
\[
\sup_{(G,p)}\big\|\mathcal{R}_\phi\!\big(\mathcal{T}_\psi(F_\theta(G),p),\varepsilon\big)-\Psi^\star(G,p)\big\|<\epsilon,
\]
and, with the Proposition above,
$\mathcal{R}_\phi\!\big(\mathcal{T}_\psi(F_\theta(G),p),\varepsilon\big)\approx\Phi(\kappa(G,p))=\Phi(G_p)$.

\paragraph{Precision.} The two claims are distinct. The reparameterisation
(the Proposition above) is exact---a set-theoretic identity,
since $\kappa$ is deterministic. The trained
$\Phi(G_p)\approx\mathcal{R}_\phi(\mathcal{T}_\psi(H,p))$ is a learned correspondence, not a
mathematical identity: it holds to the extent $F_\theta$ is lossless and
$\mathcal{T}_\psi,\mathcal{R}_\phi$ can fit $\Phi^\star\circ\kappa$. The realisability
conditions are standard---permutation-invariant set functions are universally approximable by
the Deep Sets construction~\cite{zaheerDeepSets2018}, self-attention is a universal
approximator of permutation-equivariant sequence maps~\cite{yunAreTransformersUniversal2020},
and conditioning on a perturbation rather than re-fitting on modified data is amortisation in
the sense of hypernetworks~\cite{haHyperNetworks2017}.

\paragraph{Relation to prior work.} The closest precedent is the expressivity result that a
structural graph transformation applied at preprocessing, followed by standard message
passing, is at least as expressive as the corresponding improved message-passing
algorithm~\cite{joglWeisfeilerLemanReturn2022}; that result is topological and
isomorphism-flavoured, whereas the Proposition above is a
function-class containment for feature/parametric perturbations $p$ (dose $m$, type $\tau$)
that a bare rebuilt topology cannot carry. Running a network on perturbed copies of a graph
is known to exceed 1-WL expressiveness---by random node dropout~\cite{pappDropGNN2021} or by
aggregating a policy-drawn bag of subgraphs~\cite{bevilacquaEquivariantSubgraph2022}---with
1-WL the reference power for message passing~\cite{xuHowPowerfulAre2019}. GEARS instantiates the
data-graph side biologically, materialising perturbations as edges on a gene--gene
graph~\cite{roohaniPredictingTranscriptional2024}; CGT is its amortised counterpart, keeping
the graph fixed and learning the perturbation in representation space.

\paragraph{Empirical validation.} A minimal, idealised experiment on single-gene knockout
fitness tests the equivalence directly (Supplementary
Fig.~\ref{fig:pert-equivalence-empirical}). Two capacity-matched small models are trained on
the same knockouts, gene features, and splits, differing only along the perturb-data
vs.\ perturb-representation axis: a \emph{data-perturbing} GNN that rebuilds the perturbed
graph $G_p=\kappa(G,p)$---deleting the knocked-out gene and its edges---and re-encodes,
$\hat y=\Phi(G_p)$; and a \emph{representation-perturbing} CGT that encodes the wildtype once,
$H=F_\theta(G)$, and applies the operator, $\hat y=\mathcal{R}_\phi(\mathcal{T}_\psi(H,p))$. A
single deletion gives an injective $\kappa$, so the Proposition predicts equal attainable
accuracy: the amortised model should match the data-perturbing one on held-out fitness while
encoding once rather than rebuilding and re-encoding per strain.
